AISHLING LYNCH PORTFOLIO

Puzzle Fungus



A second year assessment tasked me with creating a 3D game in Unity. I was inspired by games from the 2000s with lower poly models, and decided to combine this with a unique colour palette to create an immersive environment with mushrooms that react to the user. For example, there are mushrooms that light up and emit a sound when looked at, and others that provide a jump boost when jumped on. All models and artwork were created by me.



The game encourages the user to collect parts of a mushroom as they progress, allowing them to learn about it in the procress. There is a simple puzzle required to progress, where the user must bounce on mushrooms in the correct order. This is used as part of the prototype to demonstrate a puzzle.
